# QA Checklist (Manual + Técnico)

## Manual
- [ ] Formulario editable completo sin mouse.
- [ ] Mensajes de error visibles y asociados al input.
- [ ] Cambio de template/theme/orientation sin romper layout.
- [ ] QR visible y escaneable en móvil.
- [ ] Export PNG descarga archivo.
- [ ] Export PDF abre flujo de impresión.

## Técnico
- [ ] `npm run lint`
- [ ] `npm run build` (si el entorno tiene dependencias Vite)
- [ ] Revisar consola sin errores en flujo feliz.
